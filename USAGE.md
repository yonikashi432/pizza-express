# Golden Operator (v25) — Package
Contents:
- golden-seed-v25.yaml — Seed ready to paste.
- golden-ring.mmd — Mermaid diagram.
- golden-ring.dot — Graphviz DOT diagram.
- USAGE.md — quick commands and bootstrap.

Quick start:
- Paste `golden-seed-v25.yaml` into your config loader or agent bootstrap.
- Render `golden-ring.mmd` with Mermaid live editor or VS Code extension.
- Visualize `golden-ring.dot` with Graphviz (`dot -Tpng golden-ring.dot -o golden-ring.png`).
- Review and adapt parameters as needed for your environment.
- Use the bootstrap steps to initialize a new agent or system.

## Bootstrap Steps
1. Load the seed YAML into your agent/system.
2. Run the bootstrap steps as described in the YAML.
3. Use the diagrams to understand phase transitions and system flow.
4. Tune profiles and parameters for your use case.

## Diagram Rendering
- Mermaid: https://mermaid-js.github.io/mermaid-live-editor/
- Graphviz: https://graphviz.gitlab.io/download/
